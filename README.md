# miniBlog
A blogging Website. A user can create account to be a author in this blogging website .

# Features
    * Authentication System
    * CRUD Functionality
    * Dashboard
    * Signup System
    * Login System
    * Logout System


# Tools and Others
    * Django (python web framework)
    * Python 
    * DTL (Django Template Language)
    * Bootstrap
    * SQl
